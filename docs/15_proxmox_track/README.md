# Track Proxmox Moderno (Fasi 1-5)

Questo track affianca il percorso Vagrant storico e introduce una pipeline moderna:

1. Proxmox Foundation
2. Terraform Provisioning
3. Ansible + AWX Control Plane
4. Oracle Silent Automation
5. K3s/RKE2 Capstone

## Obiettivo

Passare da lab VM-centric a piattaforma service-centric mantenendo standard di qualità,
sicurezza e verificabilità già presenti nel repository.
