# 04. Governance & Learning
Questa directory contiene le policy di gestione, la didattica, e la compliance del repository. Raccoglie la visione ad alto livello su come un database Oracle debba essere gestito in ambiente Corporate.

## Struttura
- **`01_fondamenti_teorici/`**: Teoria dell'architettura Oracle, lifecycle di una transazione, memory SGA/PGA, glossario tecnico. Da studiare prima di avviare l'infrastruttura.
- **`02_enterprise_standards/`**: Regole d'oro per l'azienda. Standard di rilascio, Scorecard della Maximum Availability Architecture (MAA), e policy di compatibilità.
- **`03_esami_e_carriera/`**: Materiale per la preparazione delle certificazioni (1Z0-082, 1Z0-083), ripassi concettuali, e checklist operative del mestiere del DBA.
