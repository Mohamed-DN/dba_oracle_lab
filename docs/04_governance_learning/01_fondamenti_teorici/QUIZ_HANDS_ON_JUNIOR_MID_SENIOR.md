# Quiz ed Esercizi Hands-on (Junior / Mid / Senior)

## Junior

### Teoria (quiz)
1. Differenza tra SGA e PGA.
2. Scopo di redo e undo.
3. Cos'è un tablespace bigfile.

### Hands-on
- Esegui 5 query diagnostiche da `GUIDA_COMANDI_DBA`.
- Individua sessioni attive e stato listener.

### Pass/Fail
- PASS: >= 80% quiz corretto + 2/2 task pratici completati.
- FAIL: soglia inferiore o output non verificabile.

## Mid

### Teoria (quiz)
1. Flusso Data Guard transport/apply.
2. Differenza AWR vs ASH.
3. Strategia RMAN full + archivelog.

### Hands-on
- Simula analisi query lenta con runbook 05.
- Esegui check Data Guard con runbook 03.
- Esegui verifica backup runbook 02.

### Pass/Fail
- PASS: >= 75% quiz + 3/3 runbook con evidenza log.
- FAIL: mancata evidenza o runbook incompleto.

## Senior

### Teoria (quiz)
1. Trade-off protection mode Data Guard.
2. Contromisure a lock storm / CPU saturation.
3. Governance release e go/no-go.

### Hands-on
- Esegui drill switchover controllato con rollback.
- Definisci piano remediation KPI sotto target.
- Valida una guida contro standard didattico.

### Pass/Fail
- PASS: >= 70% quiz + 3/3 prove con evidenza + postmortem tecnico.
- FAIL: assenza rollback/verifica o remediation non misurabile.
