# Template Standard Guida Oracle DBA

Usa questo schema per tutte le nuove guide e per i refactor progressivi di quelle esistenti.

## 1) Obiettivo
- Cosa risolve la guida
- In quali scenari usarla

## 2) Prerequisiti
- Versione Oracle supportata/testata
- Privilegi richiesti (SYSDBA, grid, root, ecc.)
- Dipendenze (tool, patch, rete, storage)

## 3) Rischi e Impatto
- Rischi tecnici principali
- Impatto su uptime/performance
- Finestre consigliate (orario, change window)

## 4) Procedura Operativa
- Step ordinati e verificabili
- Comandi/script con input/output atteso

## 5) Rollback
- Condizioni di rollback
- Procedura di ripristino passo-passo

## 6) Validazione Finale
- Check tecnici obbligatori post-attività
- Criteri di successo/fallimento

## 7) Troubleshooting Rapido
- Errori comuni
- Diagnosi minima
- Correzione consigliata

## 8) Riferimenti
- Link interni al repository
- Riferimenti Oracle ufficiali
